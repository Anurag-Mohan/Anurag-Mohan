> **Note:** if the stats card or top-languages card above show broken/blank images while the streak card works, it's almost always the shared `github-readme-stats.vercel.app` demo instance being rate-limited or temporarily down (a very common issue, not something wrong with your username). Fixes, in order of effort:
> 1. Hard-refresh / wait — the public demo gets hit by thousands of repos and rate-limits itself; it usually recovers within a few hours.
> 2. Add `&cache_seconds=3600` (already added above) so GitHub's own image proxy caches a working copy instead of re-fetching a failing one every load.
> 3. Deploy your own free instance by forking [anuraghazra/github-readme-stats](https://github.com/anuraghazra/github-readme-stats) to Vercel, then swap the domain in your URLs for your own (e.g. `https://your-app.vercel.app/api?username=Anurag-Mohan...`). This is the most reliable long-term fix since you're no longer sharing rate limits with the whole internet.
> 4. Double-check the GitHub token/env vars if you *are* self-hosting — a missing `PAT_1` token is the #1 cause of a self-hosted instance failing.
